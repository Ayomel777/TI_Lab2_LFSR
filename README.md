# 🔐 LFSR Encryption

Десктопное приложение для шифрования файлов с использованием алгоритма LFSR (Linear Feedback Shift Register).

## Описание

Приложение шифрует и дешифрует файлы любого типа (изображения, видео, аудио, документы и др.) используя 37-битный ключ и полином x³⁷ + x¹² + x¹⁰ + x² + 1.

**Особенности:**
- Симметричное шифрование (один ключ для шифрования и дешифрования)
- Поддержка всех типов файлов
- Ключ: 37 символов (только 0 и 1)

## Требования

- Go 1.21+
- Node.js 18+
- Wails CLI

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

## Установка и запуск

```bash
# Клонирование
git clone https://github.com/username/lfsr-encryption.git
cd lfsr-encryption

# Установка зависимостей
cd frontend && npm install && cd ..

# Запуск (режим разработки)
wails dev

# Сборка
wails build
```

Собранное приложение: `build/bin/`

## Использование

1. Выберите операцию (шифрование/дешифрование)
2. Выберите файл
3. Введите 37-битный ключ (например: `1010101010101010101010101010101010101`)
4. Нажмите кнопку обработки
5. Выберите место сохранения
